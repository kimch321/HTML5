목록list 관련 태그

개요

깔끔한 정보전달을 위해 여러 요소들을 일렬로 나열한 명단이나 목록을 의미

html에서는 목록을 표현하기 위해 순서없는 목록, 순서있는 목록, 정의 목록등을 지원함

순서없는unordered 목록

명단의 요소를 기호로 나타내는 목록의 형태

ul, li등을 이용해서 표현

목록에 사용가능한 기호는 circle,square,disc등이 있음

정의 목록

용어와 그 용어에 대한 해설을 모아놓은 목록

dl태그로 목록의 시작을, dt태그로 용어의 이름을, dd태그로 해당 용어에 대한 해설을 작성

웹표준 용어

HTML
웹 사이트를 구성하는 웹 문서 작성에 사용하는 마크업 언어
CSS
html로 작성된 문서를 시각적으로 꾸밀때 사용
Javascript
html에 사용하는 객체기반 스크립트 프로그래밍 언어

목록 중첩nested하기

목록을 구성하는 요소들을 구조화structured된 정보형태로 나타낼 때 주로 사용

이런 경우 목록 속에 또 다른 목록을 포함시킴 (포함되는 목록은 ul,ol모두 가능하다.)
